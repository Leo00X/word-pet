# 快速开始工作流

适用于紧急任务或快速咨询。
> **对话语言**: 总是使用简体中文回复，包括思考过程、任务清单、代码注释和解释。忽略任何英文输出要求，除非用户指定。

---

## 🚀 30秒快速启动

### 首次对话 / 新会话

复制粘贴以下内容：

```
请阅读以下文档了解 WordParasite 项目：

@[word-pet/DEVELOPMENT.md]
@[word-pet/ARCHITECTURE.md]

这是一个基于 uni-app + Vue3 的桌面 AI 宠物应用。
核心架构是 Composable + Component 模式。

我的任务是：【描述你的任务】
```

## ⚡ 快速模板

### 添加新功能
```
@[word-pet/DEVELOPMENT.md]
@[word-pet/FEATURES.md]

我要添加【功能名】，请先制定计划
```

### 修复 Bug
```
@[word-pet/DEVELOPMENT.md]
@[word-pet/docs/maintenance/bug_prevention_guide.md]

Bug 描述：【详细描述】
复现步骤：【步骤】
```

### 代码审查
```
@[word-pet/ARCHITECTURE.md]

请审查这段代码是否符合项目规范：
【贴代码】
```
